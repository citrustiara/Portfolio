URL Notepad

2026 · GitHub · Demo

Mały notatnik, który zapisuje całą treść w adresie URL. Kopiujesz link i notatka idzie razem z nim: bez serwera, bez konta, bez bazy danych.

Mechanika

  • LZString kompresuje notatkę przed zapisaniem jej w adresie.
  • History API aktualizuje URL bez przeładowania strony.
  • Przyciski pozwalają skopiować skompresowany link albo sam tekst notatki.
