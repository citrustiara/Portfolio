WorldChem Website

2025 · GitHub · Demo

1. O projekcie

Nowoczesna i responsywna strona internetowa dla firmy WorldChem Sp. z o.o., dostawcy surowców chemicznych z Gdańska. Projekt miał na celu stworzenie profesjonalnej wizytówki online, ułatwiającej klientom zapoznanie się z ofertą i kontakt z firmą.

2. Kluczowe Funkcje

  • Dynamiczne wyszukiwanie produktów: Szybkie przeszukiwanie całej bazy produktów w czasie rzeczywistym.
  • Pełna responsywność (RWD): Zoptymalizowana dla komputerów, tabletów i telefonów.
  • Nowoczesny UI/UX: Gradienty, płynne animacje i efekty glass-morphism.
  • Formularz kontaktowy: Zintegrowany formularz z walidacją danych.
  • Zgodność z RODO: Funkcjonalny baner plików cookie.

3. Technologie

Strona została zbudowana w oparciu o czysty HTML, CSS i JavaScript, co zapewnia maksymalną wydajność i łatwość utrzymania.

  • HTML5 (struktura semantyczna)
  • CSS3 (animacje, flexbox, grid, zmienne CSS)
  • JavaScript (logika wyszukiwania, obsługa cookies, interakcje)

4. Struktura Projektu

Projekt składa się z kilku kluczowych sekcji:

  • Strona Główna: Hero section i wyróżnione produkty.
  • Oferta: Główna strona z kategoriami produktów (Odtłuszczacze, Rozpuszczalniki, Chemia profesjonalna, etc.).
  • O nas: Informacje o firmie.
  • Kontakt: Dane teleadresowe i formularz.
