WorldChem Website

2025 · GitHub · Live site

Responsive website for WorldChem Sp. z o.o., a chemical raw-materials supplier based in Gdańsk. The goal was a clear product-oriented site that helps customers browse the offer and contact the company.

Scope

  • Product catalog structure with dynamic search.
  • Responsive layouts for desktop, tablet, and mobile.
  • Contact form validation and cookie banner behavior.
  • Static HTML, CSS, and JavaScript for simple hosting and maintenance.

Sections

The site includes a homepage, product offer pages, company information, and contact areas. The implementation stays lightweight rather than introducing a framework for a mostly static business site.
