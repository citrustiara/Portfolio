URL Notepad

2026 · GitHub · Demo

Opis

Minimalistyczny notatnik, który zapisuje całą treść bezpośrednio w adresie URL. Dzięki kompresji LZString można zapisać zaskakująco dużo tekstu. Wystarczy skopiować URL żeby zachować notatkę - bez serwera, bez bazy danych.

Technologie

  • JavaScript (vanilla)
  • LZString - kompresja tekstu
  • History API - aktualizacja URL bez przeładowania

Funkcje

  • Automatyczny zapis do URL przy każdej zmianie
  • Kompresja tekstu (nawet 10x mniejszy rozmiar)
  • Całość działa w frontendzie
  • Guziki do kopiowania URL i tekstu
