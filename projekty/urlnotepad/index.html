URL Notepad

2026 · GitHub · Demo

A tiny notepad that stores the entire note inside the URL. Copy the link and the note goes with it: no server, no account, no database.

Mechanics

  • LZString compresses the note before it is written into the address.
  • The History API updates the URL without reloading the page.
  • Copy actions expose either the compressed link or the raw note text.
